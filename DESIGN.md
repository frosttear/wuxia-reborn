# 轮回江湖 — 游戏设计文档

> 武侠风轮回类文字冒险游戏

---

## 游戏概述

每世角色从 16 岁开始，随时间流逝每月触发随机事件，通过选择获取属性与奖励，磨砺武艺、结交人物，最终挑战最终Boss天魔。失败后可轮回转世，携带前世部分属性与天赋，在新的一世中变得更强。

参考游戏：虚拟人生（BitLife）——以时间流逝代替掷骰走格。

---

## 核心属性（7项）

| 属性     | 说明         | 主要影响               |
|----------|--------------|------------------------|
| 力量     | 体魄强壮     | 物理攻击、武道职业解锁 |
| 敏捷     | 身法轻盈     | 回避、速度、剑客路线   |
| 体质     | 生命韧性     | 最大血量               |
| 内力     | 灵气修为     | 内功技能、剑宗路线     |
| 悟性     | 天资聪慧     | 属性获取效率、奇遇触发 |
| 运气     | 命运眷顾     | 战斗随机浮动、稀有事件 |
| 声望     | 江湖威名     | 社交事件、大侠路线解锁 |

## 职业路线（武道路线 MVP）

```
无名小卒
  └─→ 游侠          (力量 ≥ 12)
        └─→ 剑客      (力量 ≥ 20, 敏捷 ≥ 15)
              └─→ 大侠  (力量 ≥ 35, 声望 ≥ 20)
                    └─→ 剑宗 (力量 ≥ 55, 内力 ≥ 25, 声望 ≥ 40)
```

**待扩展路线（后续版本）：**
- 仙道路线：书童 → 外门弟子 → 内门弟子 → 元婴真人
- 江湖帮派路线：帮派小弟 → 镖师 → 帮主 → 武林盟主
- 隐藏魔道路线：影卫 → 魔道修士 → 魔君（通往隐藏Boss古神）

## 事件系统

每月随机触发 1 个事件，类型与权重：

| 类型   | 权重 | 说明                           |
|--------|------|--------------------------------|
| 磨练   | 高   | 练功/读书，主动提升属性        |
| 奇遇   | 中   | 意外获得秘籍/灵药/高人指点     |
| 遭遇战 | 中   | 遭遇敌人，属性对比决定胜负     |
| 交友   | 低   | 结识/深化NPC关系，解锁事件链   |
| 机缘   | 低   | 职业晋升契机，特殊道具         |
| Boss   | 触发 | 满足条件后触发最终Boss决战     |

### 事件条件系统
每个事件支持条件筛选：
- `minAgeYears / maxAgeYears`：年龄范围
- `jobs`：当前职业限制
- `minAttributes`：最低属性要求
- `flags`：故事旗标（如 `met_wang_tie: true`）
- `npcAffinity`：NPC 好感度门槛

## NPC 关系系统

好感度 0-100，5 档标签：陌生人 / 点头之交 / 朋友 / 挚友 / 知己

### MVP 四大NPC

| NPC     | 类型     | 事件链                                      |
|---------|----------|---------------------------------------------|
| 王铁    | 师父     | 初遇 → 传基础刀法（好感≥30）→ 高级传授（≥60）→ 临终托付（≥70） |
| 李云舒  | 挚友     | 初遇 → 倾诉心事（好感≥40）→ 重逢（晚期）   |
| 赵霸天  | 宿敌     | 初遇 → 寻仇对战（多次交锋）                 |
| 神秘老者| 机缘     | 偶遇 → 传道（好感≥60）                      |

## 战斗系统

```
攻击力 = 力量×2 + 职业基础攻击 + 技能加成 + 运气浮动
防御力 = 体质×1.5 + 敏捷×0.5 + 职业基础防御

胜利：攻击力 > 敌方防御（含±随机浮动）
胜利：扣少量血量，获得声望/属性奖励
失败：扣大量血量，血量归零触发死亡/轮回
```

## 轮回系统

**触发条件**：被Boss击败 / 寿命耗尽（80岁，有天赋可延长）

**继承内容**：
1. **属性继承**：所有属性 × 10%（天赋可提升比例）
2. **传承天赋**：本世成就解锁，最多每世选 2 个携带

| 天赋     | 解锁条件           | 效果                   |
|----------|--------------------|------------------------|
| 剑心     | 达到剑客以上       | 力量/敏捷获取效率+15%  |
| 慧根     | 内力或悟性 ≥ 20    | 内力/悟性获取效率+15%  |
| 长生诀   | 活过 65 岁         | 最大血量+50，寿命+10年 |
| 宿命之力 | 轮回 ≥ 2 次        | 属性继承比例额外+5%    |
| 江湖阅历 | 声望 ≥ 15          | 初始声望+5，运气+2     |

## 技术架构

- **技术栈**：纯 HTML + CSS + JS，无框架，GitHub Pages 可直接部署
- **数据层**：JSON 文件（jobs / npcs / events / enemies）
- **存档**：localStorage 自动存档

```
index.html          主页面
css/styles.css      武侠暗色主题
js/
  character.js      角色属性、HP、职业管理
  npc.js            NPC好感度与关系系统
  combat.js         战斗结算
  rebirth.js        轮回系统与天赋
  engine.js         游戏主引擎（时间推进、事件触发）
  ui.js             UI渲染（三栏布局）
data/
  jobs.json         职业定义
  npcs.json         NPC定义
  events.json       事件库（当前38个）
  enemies.json      敌人定义
```
