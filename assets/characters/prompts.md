# NPC Character Image Prompts

AI image generation prompts for each NPC avatar. Filenames use the pattern `{npc-id-with-hyphens}.png`.

## Recommended Settings

- Steps: 30-40
- CFG scale: 7-8
- Sampler: Euler a or DPM++ 2M Karras
- Resolution: 512x512 or 512x768 (portrait)

## Shared Negative Prompt

Used for all characters:

```
nsfw, nudity, modern clothing, western features, heavy makeup, gore, deformed hands, extra fingers, blurry face, low quality, watermark, text
```
