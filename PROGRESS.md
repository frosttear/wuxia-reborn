## ✅ 已完成（v0.2.0 本次迭代）

- [x] 职业切换按钮：满足属性条件即可点击，不再仅显示"可解锁"文字

### 好感度系统升级
- [x] 好感度范围扩展为 -100 到 +100（支持负值/仇恨）
- [x] 好感度条居中设计：正值绿色向右、负值红色向左
- [x] 好感度数值显示（如 +35 / -20）
- [x] 赵霸天相遇事件改为负好感（挺身而出 -20，暗中观察 -5）
- [x] 所有NPC增加负值标签（结怨/不快/宿敌/死敌/不共戴天等）

### 职业技能系统
- [x] 职业技能对象化（id/name/desc/branch/bonuses/special）
- [x] 晋升职业时自动学习该职业被动技能，同分支累积保留
- [x] 跨分支切换时清除旧分支技能并提示
- [x] 攻防计算改为累加 `char.learnedSkills` 而非仅靠当前职业
- [x] 特殊技能：侠义之心（声望/10=攻击）、以气御剑（内力/10×3）
- [x] 已学技能列表显示在左侧面板（含攻/防加成标注）

### 页面布局重构
- [x] 职业路线移至左侧面板（属性正下方），无需跨栏对比
- [x] 职业需求改为 `当前值/需求值 ✓/✗` 标签（绿色=满足/红色=不足）
- [x] 右侧面板改为：江湖人物 + 传承天赋
- [x] 左侧面板加宽至260px容纳更多信息

### 悟性/运气系统
- [x] 悟性：每10点提升5%属性学习效率（所有正向属性获取均受益）
- [x] 悟性：提升奇遇/机缘事件触发权重（每5点+1权重）
- [x] 运气：每次属性事件有运气/200概率触发双倍收益（日志提示）
- [x] 运气：战斗闪避（运气/200概率减伤50%）
- [x] 运气：降低遭遇战触发概率
- [x] 界面显示三项派生数值：📖学习效率 / 💨闪避率 / ✨幸运率

### 战斗系统升级
- [x] 敌人属性随玩家力量动态缩放（每10点力量涨一档）
- [x] 新增「江湖挑战者」敌人（高缩放系数，赢/输均涨悟性）
- [x] 战斗选项显示实际敌人属性（含缩放后）+ 胜率预估（蒙特卡洛200次）
- [x] 胜率颜色标识：🟢≥70% / 🟡40-70% / 🔴<40%
- [x] 选项奖励预览（属性/气血/好感度/战斗对手一目了然）

### 数值平衡
- [x] 天魔防御 65→228，攻击 85→130（约需3次轮回方可击败）
- [x] Boss触发条件：需达到剑宗 + 年龄≥45岁（原：大侠/剑宗 + 40岁）
