## ✅ 已完成（v0.8.0）

### 战斗系统平衡调整
- [x] **删除「巧攻」行动**：对手不会防御，巧攻无存在意义，从 `index.html`、`combat.js`、测试文件中完全移除
- [x] **「刚攻」→「强攻」，「化解」→「招架」**：按钮名称与工具提示同步更新
- [x] **强攻伤害公式重新平衡**：
  - 旧：`atk × 1.2 - def × 0.5`（ATK 效率是 DEF 的 2.4 倍，力量碾压防御）
  - 新：`atk × lv - def × 0.75`（1:0.75 ATK/DEF 比，轻微攻强防弱，更符合武侠风格）
  - `lv`：运气浮动系数 `1 ± (luck/200)`，幸运值越高波动越大
- [x] **职业主动技能威力提升**（技能应明显强于普通强攻）：
  - 游侠「破势一击」power：1.8 → 2.2
  - 剑客「剑气三连」power/hit：0.65 → 0.85
  - 剑侠「侠气凌云」power：2.5 → 3.0
  - 剑宗「剑意通天」power：3.0 → 3.8
- [x] **多段技能伤害修正**：防御削减改为对总伤一次性计算（原为每 hit 各扣防御，导致多段技能弱于单段强攻）
