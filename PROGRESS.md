# 轮回江湖 — 开发进度

## 当前版本：v0.4.2

---

## ✅ 已完成（v0.1.0 初始MVP）

### 数据层
- [x] `data/jobs.json` — 5个武道职业（无名小卒/游侠/剑客/剑侠/剑宗）
- [x] `data/npcs.json` — 4个NPC（王铁/李云舒/赵霸天/神秘老者）
- [x] `data/enemies.json` — 6类敌人（含最终Boss天魔）
- [x] `data/events.json` — 38个事件（磨练12/奇遇7/遭遇战7/交友8/机缘4）

### 核心系统
- [x] 角色系统：7项属性、HP计算、职业解锁
- [x] NPC关系系统：好感度0-100，5档标签，多段事件链
- [x] 战斗系统：攻防对比 + 运气浮动，胜负分支
- [x] 轮回系统：10%属性继承 + 5种传承天赋（每世最多选2个）
- [x] 游戏引擎：月度推进、加权随机事件、条件筛选、Boss触发
- [x] 存档系统：localStorage 自动存档/读档

### UI（v0.1.0）
- [x] 开始界面（输入名字/继续前世）
- [x] 三栏布局：角色面板 / 事件日志 / 人物关系+职业
- [x] 事件展示与选择按钮
- [x] HP血条、属性面板、战斗力显示
- [x] NPC好感度进度条
- [x] 职业树（含解锁状态与切换）
- [x] 轮回结算弹窗（展示本世成就 + 天赋选择）
- [x] 胜利弹窗
- [x] 手动推进 + 自动推进（可暂停）

## ✅ 已完成（v0.4.1）

- [x] 职业晋升新增事件门槛（`requiredFlags`）
  - 剑侠：需「武林认可」事件（`hero_recognized`）
  - 剑宗：需「剑客传承」事件（`sword_legacy`）
- [x] `character.meetsJobRequirements()` 支持 `requiredFlags`
- [x] 职业面板显示事件要求标签（金色斜体，未满足显示「（事件）」）
- [x] 真结局系统（类P5 True Ending）：
  - 普通结局：击败天魔 → 天下太平
  - 真结局：击败天魔 + `jade_tablet_awakened` → 剑魂现身
  - 击败剑魂 → ✦ 真·天下太平 ✦
  - 败于剑魂 → 轮回「功亏一篑，败于剑魂」
- [x] 25岁生辰玉牌异变事件（设置 `jade_tablet_awakened` flag）
- [x] `triggerVictory(isTrueEnding)` 双结局分支
- [x] `showVictoryScreen(char, isTrueEnding)` 双结局界面
- [x] 职业改名：大侠 → 剑侠
