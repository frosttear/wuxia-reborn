# 轮回江湖

> 武侠风轮回类文字冒险游戏 · 纯 HTML/CSS/JS · GitHub Pages 可直接部署

**[在线游玩 →](https://frosttear.github.io/wuxia-reborn/)**

---

## 游戏概述

每世角色从 **15 岁**开始，每月推进一次，触发随机事件，通过选择获取属性与奖励，磨砺武艺、结交人物。**20岁生辰**，天魔如约而至——无论强弱，这场决战无可回避。失败后轮回转世，携带前世部分属性与天赋，在新的一世中变得更强。

参考：虚拟人生（Virtual Life）——以时间流逝代替掷骰走格，每次选择都有重量。

## 核心属性（7项）

| 属性 | 说明 | 主要影响 |
|------|------|----------|
| 力量 | 体魄强壮 | 物理攻击、武道职业解锁 |
| 敏捷 | 身法轻盈 | 回避、速度、剑客路线 |
| 体质 | 生命韧性 | 最大血量 |
| 内力 | 灵气修为 | 内功技能、剑宗路线 |
| 悟性 | 天资聪慧 | 属性获取效率、奇遇触发 |
| 运气 | 命运眷顾 | 战斗随机浮动、稀有事件 |
| 声望 | 江湖威名 | 社交事件、剑侠路线解锁 |

## 职业路线（武道路线）

```
无名小卒
  └─→ 游侠  (力量 ≥ 5)
        └─→ 剑客  (力量 ≥ 10, 敏捷 ≥ 7)
              └─→ 剑侠  (力量 ≥ 22, 敏捷 ≥ 12, 声望 ≥ 12 + 武林认可事件)
                    └─→ 剑宗  (力量 ≥ 32, 内力 ≥ 14, 敏捷 ≥ 16, 声望 ≥ 20 + 剑意传承事件)
```

**事件门槛**：剑侠需完成「武林认可」事件（`hero_recognized` flag）；剑宗需完成「剑客传承」事件（`sword_legacy` flag）。单纯堆属性不足以晋升。

**待扩展路线：**
- 仙道路线：书童 → 外门弟子 → 内门弟子 → 元婴真人
- 江湖帮派路线：帮派小弟 → 镖师 → 帮主 → 武林盟主
- 隐藏魔道路线：影卫 → 魔道修士 → 魔君

## 时间与生命周期

- **起点**：15岁（ageMonths = 180）
- **终点**：20岁生辰（ageMonths = 240）天魔降临
- **安全上限**：maxAgeMonths = 246
- **事件频率**：每月推进，每次触发一个随机事件
- **单属性上限**：每次事件最多 +2（防止属性膨胀）

## 生日系统

每12个月触发一次生日事件，专属台词 + 属性奖励，之后继续触发当月随机事件（包含专属生日事件池）。

| 年龄 | 内容 |
|------|------|
| 16 | 初踏江湖感慨 |
| 17 | 少年老成，悟性+1 |
| 18 | 已能独当一面，力量+1 声望+1 |
| 19 | 「距天魔之约只剩最后一年」，内力+1 |
| **20** | **天魔降世**（无条件触发） |

**生日专属事件**（权重40，优先触发）：
- 「生辰感悟」：独自冥想，悟性/内力+1~2
- 「老镖师庆生」：与王铁共饮，好感+8~10
- 「云舒相贺」：李云舒送来点心，好感+8~10
- 「老者生辰赠言」：神秘老者夜访，好感+8~10

## 事件系统

| 类型 | 权重 | 说明 |
|------|------|------|
| 磨练 | 高 | 练功/读书，主动提升属性 |
| 奇遇 | 中 | 意外获得秘籍/灵药/高人指点 |
| 遭遇战 | 中 | 遭遇敌人，回合制战斗 |
| 交友 | 低 | 结识/深化NPC关系，解锁事件链 |
| 机缘 | 低 | 职业晋升契机、特殊道具 |
| Boss | 触发 | 生日系统触发，不走随机池 |

### 事件条件
- `minAgeYears / maxAgeYears`：年龄范围
- `jobs / notJobs`：职业限制
- `minAttributes`：最低属性要求
- `flags`：故事旗标（如 `sword_legacy: true`，`_is_birthday: true`）
- `npcAffinity`：NPC 好感度门槛

## NPC 关系系统

好感度 -100 到 +100，正负双向，居中条形图显示。

### NPC 一览

| NPC | 类型 | 羁绊章节 |
|-----|------|---------|
| 王铁 | 师父 | 第1章（好感≥25）→ 第2章（≥50）→ 第3章（≥75，≥19岁） |
| 李云舒 | 挚友 | 第1章（好感≥20）→ 第2章（≥45）→ 第3章（≥70，≥18岁） |
| 神秘老者 | 机缘 | 第1章（好感≥25）→ 第2章（≥50）→ 第3章（≥75）→ **真结局钥匙** |
| 寒霜 | 江湖人 | 独立事件链 |
| 苏清 | 江湖人 | 独立事件链 |
| 赵霸天 | 宿敌 | 好感负值，多次交锋 |

### 拜访机制

满足条件时**随时可拜访**已结识的 NPC：
- **好感达标** → 触发完整羁绊章节事件
- **好感不足** → 闲聊拜访，好感 **+3**，并提示距下一章还差多少
- 每次拜访消耗 **1 个月**

## 战斗系统（回合制）

```
攻击力 = 力量×0.5 + 职业基础攻击 + 技能加成
防御力 = 体质×1.5 + 敏捷×0.5 + 职业基础防御

闪避率 = 敏捷/250 + 运气/200（上限 25%）
暴击率 = 悟性/250 + 运气/200 + 内力/250（上限 35%）
```

每回合：出招 / 防御（伤害减半）/ 逃跑（25% 成功率）

战斗选项预显示敌人属性（动态缩放）与胜率（蒙特卡洛估算，🟢🟡🔴色标）。

## 轮回系统

**触发条件**：被 Boss 击败 / 战斗死亡 / 寿命耗尽

**继承内容**：
1. **属性继承**：所有属性 × 10%（天赋可提升比例）
2. **出生月份**：同一命运，同一生辰
3. **羁绊记忆**：前世羁绊等级 → 下一世初始好感加成
4. **传承天赋**：本世成就解锁，每世最多选 2 个

| 天赋 | 解锁条件 | 效果 |
|------|---------|------|
| 剑心 | 达到剑客以上 | 力量/敏捷获取效率+15% |
| 慧根 | 内力或悟性 ≥ 20 | 内力/悟性获取效率+15% |
| 长生诀 | 活过 38 岁 | 最大血量+50，寿命+5年 |
| 宿命之力 | 轮回 ≥ 2 次 | 属性继承比例额外+5% |
| 江湖阅历 | 声望 ≥ 15 | 初始声望+5，运气+2 |

## Boss 设计与结局系统

### 普通结局
- **天魔**：20岁生辰无条件触发
- 击败天魔 → **「天下太平」** 普通结局

### 真结局路径
```
神秘老者第3章（elder_revelation）
  → 击败天魔
    → 「你以为，一切终于结束了……」
      → 剑魂（sword_soul）现身
          ├─ 胜 → ✦ 真·天下太平 ✦
          └─ 败 → 轮回「功亏一篑，败于剑魂」
```

### 隐藏 Boss：剑魂
- 出现时机：打败天魔后（非随机，非生日触发）
- 老者昔年封印的未竟之敌，胜利才是真正的结局

## 技术架构

- **技术栈**：纯 HTML + CSS + JS，无框架，GitHub Pages 直接部署
- **数据层**：JSON 文件（jobs / npcs / events / enemies / bonds）
- **存档**：localStorage 自动存档

```
index.html          主页面（出生月份选择 + 游戏界面）
css/styles.css      武侠暗色主题
js/
  character.js      角色属性、HP、职业管理、出生月份加成
  npc.js            NPC好感度与关系系统
  combat.js         回合制战斗
  rebirth.js        轮回系统与天赋
  engine.js         游戏主引擎（月度推进、生日系统、事件触发、真结局）
  ui.js             UI渲染（三栏布局 + 移动端Tab + 拜访面板）
data/
  jobs.json         职业定义（含 requiredFlags）
  npcs.json         NPC定义
  events.json       事件库（含生日专属事件）
  enemies.json      敌人定义（含天魔/剑魂）
  bonds.json        羁绊事件定义
```
