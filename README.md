## 游戏概述

每世角色从 **15 岁**开始，每月推进一次，触发随机事件，通过选择获取属性与奖励，磨砺武艺、结交人物。**20岁生辰**，天魔如约而至——无论强弱，这场决战无可回避。失败后双鱼玉佩触发**世界线回溯**——时间倒流，你带着这一世的记忆回到起点的另一条世界线，在新的时间线中变得更强。

参考：虚拟人生（Virtual Life）——以时间流逝代替掷骰走格，每次选择都有重量。

## 职业路线（武道路线）

```
无名小卒
  └─→ 游侠  (力量 ≥ 12)
        └─→ 剑客  (力量 ≥ 18, 敏捷 ≥ 12)
              └─→ 剑侠  (力量 ≥ 26, 敏捷 ≥ 15, 声望 ≥ 18 + 武林认可任务)
                    └─→ 剑宗  (力量 ≥ 36, 内力 ≥ 18, 敏捷 ≥ 20, 声望 ≥ 25 + 剑意传承任务)
```

**任务门槛**：剑侠需完成「侠名之路」任务链（`hero_recognized` flag）；剑宗需完成「剑道传承」任务链（`sword_legacy` flag）。高阶职业通过专属任务线解锁，包含剧情、属性门槛和战斗考验。

## 时间与生命周期

- **起点**：15岁（ageMonths = 180）
- **终点**：20岁生辰（ageMonths = 240）天魔降临
- **安全上限**：maxAgeMonths = 246
- **事件频率**：每月推进，每次触发一个随机事件
- **单属性上限**：每次事件最多 +2（防止属性膨胀）

每12个月触发一次生日事件，专属台词 + 属性奖励，之后继续触发当月随机事件（包含专属生日事件池）。

| 年龄 | 内容 |
|------|------|
| 16 | 初踏江湖感慨 |
| 17 | 少年老成，悟性+1 |
| 18 | 已能独当一面，力量+1 声望+1 |
| 19 | 「距天魔之约只剩最后一年」，内力+1 |
| **20** | **天魔降世**（无条件触发） |

**生日专属事件**（权重40，优先触发）：
- 「生辰感悟」：独自冥想，悟性/内力+1~2
- 「老镖师庆生」：与王铁共饮，好感+8~10
- 「云舒相贺」：李云舒送来点心，好感+8~10
- 「老者生辰赠言」：神秘老者夜访，好感+8~10

## 战斗系统（回合制）

### 基础数值公式

```
攻击力 = floor(力量×0.7 + 敏捷×0.3) + 职业基础攻击 + 技能加成 + 被动加成
防御力 = floor(体质×0.7 + 敏捷×0.3) + 职业基础防御 + 技能加成

闪避率 = 运气/120 + 敏捷/250（上限 25%）
会心率 = 运气/120（上限 35%）
气盾   = floor(内力/8) + 天赋/被动加成（每次受击扁平减伤）
技能增幅 = 内力% + 天赋加成（如30内力 = +30%技能伤害）
```

### 防御穿透（固定穿透制）

```
defPen = floor(攻击方攻击力 × 0.3)
effectiveDef = max(0, 防御方防御力 - defPen)
```

双向对称：玩家攻击敌人、敌人攻击玩家均使用同一公式。

### 伤害计算

**强攻伤害**：
```
baseDmg = max(1, floor(ATK × lv) - effectiveDef - 敌方气盾)
```
- `lv`：运气浮动系数 `1 ± (luck/200)`
- 会心一击 ×1.5

**技能增幅抑制**（IF dampening）：
```
effectiveAmp = rawAmp × 100 / (100 + 对方内力 × 3)
```
高内力敌人可抑制你的技能增幅；你的内力也抑制敌人技能。

**反击伤害**（破招成功时）：
```
counterDmg = max(1, floor(ATK × counterMult) - effectiveDef - 敌方气盾)
```
- 普通反击 `counterMult = 0.75`，洞察反击 `counterMult = 1.0`
- 技能就绪时以技能代替基础反击（含多段/震慑等特效）

**防御减伤**（随攻防比缩放）：
```
powerRatio = min(1, 防御力 / max(1, 敌方攻击力))
defCap = min(0.75, powerRatio × 0.75 + 0.10)
```
- 防御 vs 重攻：减伤 min(75%, defCap)
- 防御 vs 快攻：减伤 min(50%, defCap)
- 蓄势减伤：min(55%, defCap)
- 洞察快攻时防御减伤突破 defCap：至少 defCap+15%（上限80%）

### 行动选项

| 行动 | 效果 | 气力 | 备注 |
|------|------|------|------|
| ⚔ 强攻 | 造成伤害，可暴击（×1.5） | +2 | 固定穿透30%ATK |
| 🛡 防御 | 重攻减伤75%，快攻减伤50% | — | 识破快攻时减伤提升至80% |
| ⚡ 破招 | 克制重攻：反击+仅承受20%伤害 | +1 | 破招失败：×1.15伤害 |
| 🔮 蓄势 | 减伤55%，积攒气力 | +3 | 用于蓄满技能槽 |
| 🏃 逃跑 | 25%成功率，每次失败+15% | — | 失败时受全额伤害 |

- **技能**：气力满后下回合自动发动（强攻/蓄势时触发），冷却3回合
- **破招反击**时若技能就绪，以技能作为反击招式发动（含多段连击等），不浪费技能

### 敌方行为

敌人每回合随机选择**重攻**或**快攻**：
- 气血 > 35% 时：重攻概率 45%
- 气血 ≤ 35% 时：重攻概率 65%（孤注一掷）
- ⚠️ 敌人没有「破招」行动

### 敌方意图识别

每回合结束时，根据悟性判断是否能识破敌人下回合出招意图：

```
识破概率 = min(80%, 80% × ln(1 + 悟性 / (敌方悟性 + 20)))
```

| 识破结果 | 显示 |
|---------|------|
| 识破（accurate） | 真实意图描述，100% 准确 |
| 看不清（unreadable） | 模糊迷惑文字 |
| 无相剑意被动 | 永远 100% 识破（perfect） |

**意图应对加成**（仅在 accurate/perfect 时生效）：

| 识破结果 | 应对行动 | 加成效果 |
|---------|---------|---------|
| 快攻来袭 | 防御 | 减伤提升至 80%（普通防御快攻仅50%） |
| 重攻来袭 | 破招 | 反击倍率提升至 1.0×（普通 0.75×），额外+1气力 |

### 职业主动技能

每个职业拥有一个主动技能，战斗面板**全程显示**技能名称、效果及剩余所需气力：

| 状态 | 显示 |
|------|------|
| 蓄势中 | 「技能名」效果描述　还差 X 气 |
| 可发动 | ⚡「技能名」蓄势完成，下回合自动发动！（金色闪烁） |
| 冷却中 | 「技能名」效果描述　冷却中（剩 X 回合） |

### 战斗预览系统

点击战斗动作按钮后展开预览栏，显示预估伤害/减伤/气力变化等详细数值；再次点击同一按钮执行行动。默认选中强攻。

### 其他机制

- **快速战斗**：胜率接近100%时可一键跳过，蒙特卡洛50局模拟直接结算
- **战斗状态持久化**：战斗进行中刷新页面不丢失状态（localStorage）
- **受伤状态**：战斗失败后进入重伤，强制参与「养伤」事件共 **3个月**，第4个月自动痊愈。王铁被动可将恢复期缩至 2 个月
- **敌人属性缩放**：敌人攻防HP随角色年龄动态成长（`tier = 年龄 - 15`）

**每月气血回复**（百分比制）：
- 基础：15% 最大 HP
- 体质：每点 +0.5% 最大 HP（20体质 = +10%）
- 内力：每点 +0.5% 最大 HP（20内力 = +10%）
- 苏青被动「青心丹药」：额外回复 100% 损失气血（几乎一月回满）

## 世界线回溯系统

**设定**：双鱼玉佩在佩戴者濒死时触发时间倒流，将意识连同记忆送回另一条世界线的起点。这不是转世，而是世界线的分岔——你记得「未来」发生过的一切。

**触发条件**：被 Boss 击败 / 战斗死亡 / 寿命耗尽

**继承内容**：
1. **属性继承**：所有属性 × 10%（天赋可提升比例）
2. **出生月份**：同一命运，同一生辰
3. **世界线记忆**：上条世界线的羁绊等级 → 新世界线初始好感加成，且已结识NPC无需再次相识

## Boss 设计与结局系统

### 普通结局
- **天魔**：20岁生辰无条件触发
- 击败天魔 → **「天下太平」** 普通结局

### 真结局路径
```
凌雪羁绊第5章 + 其余所有NPC羁绊满级（allBondsComplete）
  → 击败天魔（20岁生辰）
    → 「你以为，一切终于结束了……」
      → 剑魂（sword_soul）现身
          ├─ 胜 → ✦ 真·天下太平 ✦
          └─ 败 → 世界线回溯「功亏一篑，败于剑魂」
```

### 隐藏 Boss：凌雪（羁绊解锁）
- 出现时机：凌雪羁绊第5章（好感≥92，≥17岁结识）
- 天魔首席弟子，以冰属剑意对决，胜利解锁进入剑魂的资格

### 隐藏 Boss：剑魂
- 出现时机：击败天魔后，且所有NPC羁绊已满级（含凌雪）
- 上古剑修残留意志，曾创造噬魂真经引诱陆无归（天魔）为其容器，被沈玄清（神秘老者）以七成修为封印于双鱼玉佩中
- 弱点是"不懂人心"——胜利需要的不只是武力
